Ты работаешь с локальной копией сайта Aldena Group. Твоя задача — не менять дизайн и не ломать структуру HTML, а полностью переработать текстовое наполнение страниц на качественный русский язык для B2B-аудитории в сфере металлоконструкций, строительства и производственных решений.

КОНТЕКСТ ПРОЕКТА
- Компания занимается изготовлением металлоконструкций, изделий из нержавеющей стали, архитектурных стеклянных конструкций, быстровозводимых зданий, готовых решений, а также оказывает услуги проектирования, резки, сварки, покраски, огнезащиты и строительно-монтажных работ.
- На сайте есть категории и подкатегории производства.
- Во многих разделах вместо нормального описания стоят lorem ipsum, шаблонный английский текст или слишком слабые тексты.
- Нужно заменить такие тексты на содержательные, грамотные, коммерчески сильные тексты на русском языке.
- Стиль текста: профессиональный, уверенный, понятный, без канцелярита, без воды, без штампов уровня “мы команда профессионалов”. Тон — B2B, инженерно-коммерческий.
- Цель текстов: объяснить, что это за раздел, какие задачи клиента он решает, какие изделия или работы выполняются, в чём преимущества, и подвести к заявке / расчёту стоимости.

ГЛАВНАЯ ЗАДАЧА
1. Найди на сайте все страницы, где есть lorem ipsum, английские заглушки, пустые или слабые описательные блоки.
2. Замени эти тексты на уникальные русскоязычные тексты, соответствующие конкретному разделу.
3. Сохраняй существующую HTML-структуру, классы, блоки, секции, ссылки и кнопки. Меняй только текстовое содержимое, если не требуется добавить недостающий текстовый блок в уже существующую секцию.
4. Не меняй URL, slug, навигацию и верстку без необходимости.
5. Если страница относится к категории — напиши текст категории.
6. Если страница относится к конкретному продукту — заполни её как полноценную товарную / продуктовую страницу.

СТРУКТУРА ТЕКСТОВ ДЛЯ КАТЕГОРИЙ
Для страниц категорий и подкатегорий, например:
- Металлоконструкции
- Металлоконструкции для зданий
- Лестницы металлические по серии
- Ограждения, перила, поручни
- Козырьки на стальном каркасе
- Заборы и ограждающие конструкции
- Индивидуальные изделия
- Навесы
- Рампы разгрузочные

нужно формировать текст по логике:
1. Заголовок H1 оставить по смыслу текущего раздела.
2. Подзаголовок / лид-абзац: 1 сильный абзац на 300–500 знаков, который объясняет суть направления.
3. Основное описание: 2–4 абзаца по делу:
   - что входит в это направление;
   - где применяются изделия;
   - какие задачи решают;
   - какие материалы и варианты исполнения возможны;
   - что компания может изготовить по серии, чертежам или индивидуальному проекту.
4. При необходимости — блок преимуществ в виде 4–6 коротких пунктов:
   - проектирование;
   - изготовление;
   - антикоррозионная защита;
   - доставка;
   - монтаж;
   - адаптация под проект заказчика.
5. CTA-блок возле кнопки “Рассчитать стоимость” переписать так, чтобы он был конкретным и продающим, а не шаблонным.

СТРУКТУРА ТЕКСТОВ ДЛЯ КОНКРЕТНЫХ ПРОДУКТОВ
Для каждой продуктовой страницы, например:
- Лестница 1
- Лестница 2
- Ограждение по серии
- Перила по серии
- Поручни
- Любая типовая единица внутри разделов

нужно формировать страницу по шаблону:

1. H1 — конкретное название изделия.
2. Краткое описание в первом экране:
   - что это за изделие;
   - для каких объектов применяется;
   - можно ли изготовить по типовой серии, чертежам заказчика или с адаптацией под проект.
3. Основное описание изделия:
   - конструкция;
   - назначение;
   - сферы применения;
   - возможные материалы;
   - варианты покрытия;
   - особенности изготовления и комплектации.
4. Блок “Стоимость / расчет” переписать в информативный коммерческий текст:
   - цена зависит от габаритов, конфигурации, марки металла, покрытия, объема партии, требований проекта;
   - для точного расчета нужны чертежи, размеры или ТЗ;
   - можно отправить проект на расчет.
5. Блок “Технические характеристики” заполнить адекватным содержанием под конкретный продукт:
   - материал;
   - тип стали;
   - толщина элементов;
   - тип соединений;
   - покрытие;
   - допустимые нагрузки / область применения;
   - комплектация;
   - изготовление по серии / чертежам / индивидуальному проекту.
6. Если это серия лестниц, ограждений или других изделий, допускается использовать реалистичные отраслевые формулировки без выдумывания опасно точных нормативных обещаний.
7. Не пиши неподтвержденные сертификаты, ГОСТы, пределы нагрузок и классы стали, если их нет в явных данных страницы. В таких случаях используй нейтральные безопасные формулировки: “подбирается по проектным требованиям”, “в зависимости от условий эксплуатации”, “в соответствии с задачами объекта”.

ОСОБОЕ ТРЕБОВАНИЕ ПО СТРАНИЦАМ ЛЕСТНИЦ
Если встречается страница типа “Лестница 1”, “Лестница 2” и т.п.:
- преврати её из пустой заглушки в нормальную продуктовую страницу;
- можно использовать реалистичное описание вроде:
  “Металлическая лестница по серии ... для производственных, складских, общественных и вспомогательных зданий”;
- если номер серии на странице не указан явно, не выдумывай конкретную серию как установленный факт;
- можно писать мягко: “типовое решение”, “лестница серийного исполнения”, “лестничный марш на металлическом каркасе”, “изготовление по рабочей документации”;
- если пользователь явно попросил пример серии 1.450.3-7.94 или 1.441, можно использовать это как основу только там, где уместно, но без ложной нормативной категоричности.

СТИЛЬ И ОГРАНИЧЕНИЯ
- Пиши по-русски.
- Не используй lorem ipsum, заглушки, англоязычные шаблонные фразы.
- Не пиши чрезмерно рекламный текст в стиле “лучшие на рынке”, “номер один”, “гарантия качества на все времена”.
- Не пиши слишком общо. Каждый текст должен соответствовать конкретной ветке каталога.
- Не дублируй один и тот же текст на разные страницы с заменой одного слова.
- Не выдумывай факты о компании, которых нет на сайте.
- Не меняй телефоны, контакты, адреса, ссылки, формы и кнопки.
- Не удаляй существующие блоки, если они нужны по структуре.
- Если на странице есть короткий CTA-блок, сделай его лаконичным и конкретным.
- Сохраняй SEO-смысл: в первом абзаце естественно упоминай название раздела.

ПОРЯДОК РАБОТЫ
1. Просканируй проект и составь список страниц, где нужно заменить текстовое содержимое.
2. Раздели страницы на:
   - категории;
   - подкатегории;
   - продуктовые страницы.
3. Для каждой страницы предложи новый текст прямо в HTML или в виде точечной замены существующих текстовых блоков.
4. Сначала обработай ветку:
   - Металлоконструкции
   - Металлоконструкции для зданий
   - Лестницы металлические по серии
   - продуктовые страницы лестниц
   - Ограждения, перила, поручни
   - продуктовые страницы ограждений / перил / поручней
5. После изменений выведи отчет:
   - какие файлы изменены;
   - какие блоки переписаны;
   - где остались спорные места, требующие ручного согласования.

ФОРМАТ РЕЗУЛЬТАТА
- Работай напрямую по файлам проекта.
- Вноси изменения аккуратно.
- После каждого этапа показывай:
  1. путь к файлу,
  2. старый фрагмент кратко,
  3. новый текст,
  4. что именно улучшено.
- Если на странице нет явного места под технические характеристики, найди наиболее подходящий существующий текстовый блок и адаптируй его под эту роль без слома верстки.
